--- R0.9: LangGraph Resume API Confirmed ---

CURRENT (WRONG):
  result = await self._graph.ainvoke(
      input=None,
      config=config,
      interrupt_resume={"approved": True, "approval_id": approval_id},
  )

CORRECT API:
  from langgraph.graph.state import Command
  
  result = await self._graph.ainvoke(
      Command(resume={"approved": True, "approval_id": approval_id}),
      config=config,
  )

Source: https://langchain-ai.github.io/langgraph/how-tos/human_in_the_loop/wait-user-input/

The `Command(resume=...)` value becomes the return value of the
interrupt() call in the _approval_gate node, which then routes to
execute_approved_tools.
