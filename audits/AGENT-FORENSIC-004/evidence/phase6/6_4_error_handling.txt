2. VALIDATION ERROR HANDLING
----------------------------
Source: /home/zaks/zakops-agent-api/apps/agent-api/app/main.py (Lines 87-116)

Custom RequestValidationError handler:
```python
@app.exception_handler(RequestValidationError)
async def validation_exception_handler(request: Request, exc: RequestValidationError):
    # Log the validation error
    logger.error(
        "validation_error",
        client_host=request.client.host if request.client else "unknown",
        path=request.url.path,
        errors=str(exc.errors()),
    )

    # Format the errors to be more user-friendly
    formatted_errors = []
    for error in exc.errors():
        loc = " -> ".join([str(loc_part) for loc_part in error["loc"] if loc_part != "body"])
        formatted_errors.append({"field": loc, "message": error["msg"]})

    return JSONResponse(
        status_code=status.HTTP_422_UNPROCESSABLE_ENTITY,
        content={"detail": "Validation error", "errors": formatted_errors},
    )
```

Features:
- Custom error formatting (removes "body" prefix)
- User-friendly field -> message format
- Logs full error details server-side
- Client receives sanitized error only

3. RATE LIMIT ERROR HANDLING
----------------------------
Source: /home/zaks/zakops-agent-api/apps/agent-api/app/core/middleware/rate_limiter.py (Lines 80-115)

```python
async def rate_limit_exceeded_handler(request: Request, exc: RateLimitExceeded) -> JSONResponse:
    return JSONResponse(
        status_code=429,
        content={
            "error": "rate_limit_exceeded",
            "detail": "Too many requests. Please slow down.",
            "retry_after": getattr(exc, "retry_after", 60),
        },
        headers={
            "Retry-After": str(getattr(exc, "retry_after", 60)),
            "X-RateLimit-Limit": str(getattr(exc, "limit", "unknown")),
        },
    )
```

Features:
- Standard 429 status code
- Helpful Retry-After header
- No internal rate limit details exposed
