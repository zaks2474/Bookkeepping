2. CLIENT IDENTIFICATION
------------------------
From rate_limiter.py (Lines 19-42):
```python
def get_client_identifier(request: Request) -> str:
    ip = get_remote_address(request)
    auth_header = request.headers.get("Authorization", "")

    if auth_header.startswith("Bearer "):
        token_hash = hash(auth_header[7:]) % 10**8
        return f"{ip}:auth:{token_hash}"

    return ip
```

Benefits:
- Unauthenticated users rate-limited by IP
- Authenticated users get separate rate limit buckets
- Prevents authenticated users from being blocked by anonymous traffic
