2. METRICS ENDPOINT ACCESS CONTROL
----------------------------------
From metrics.py (Lines 45-60):
```python
def setup_metrics(app):
    # Only expose /metrics in development or when explicitly enabled (UF-005)
    if settings.ENVIRONMENT == Environment.DEVELOPMENT or \
       os.getenv("ENABLE_API_DOCS", "false").lower() == "true":
        app.add_route(
            "/metrics",
            lambda _: Response(generate_latest(), media_type=CONTENT_TYPE_LATEST),
            methods=["GET"],
        )
```

Security feature (UF-005):
- /metrics endpoint ONLY available in development
- Production requires explicit ENABLE_API_DOCS=true
- Prevents accidental metrics exposure
