AGENT-FORENSIC-001 Phase 1 — Step 1.3: Request/Response Schema Audit
=====================================================================
Date: 2026-02-02
Source: OpenAPI spec + live testing

=== POST /api/v1/auth/register ===
Content-Type: application/json
Request: {"email": str, "password": str (SecretStr)}
Response 200: {"id": int, "email": str, "token": str}
Response 422: {"detail": "Validation error", "errors": [...]}

=== POST /api/v1/auth/login ===
Content-Type: application/x-www-form-urlencoded
Request: username=str&password=str&grant_type=password
Response 200: {"access_token": str, "token_type": "bearer", "expires_at": datetime}

=== POST /api/v1/auth/session ===
Content-Type: application/json
Auth: Bearer JWT
Response 200: {"session_id": str(uuid), "name": str, "token": Token}

=== GET /api/v1/auth/sessions ===
Auth: Bearer JWT
Response 200: [{"session_id": str, "name": str, "token": Token}, ...]

=== POST /api/v1/chatbot/chat ===
Content-Type: application/json
Auth: Bearer JWT or X-Service-Token
Request: {"messages": [{"role": str, "content": str}]}
Response 200: {"messages": [{"role": str, "content": str}]}

=== POST /api/v1/chatbot/chat/stream ===
Content-Type: application/json → text/event-stream
Auth: Bearer JWT or X-Service-Token
Request: {"messages": [{"role": str, "content": str}]}
Response: SSE stream with data: {"content": str, "done": bool}

=== POST /api/v1/agent/invoke ===
Content-Type: application/json
Auth: NONE
Request: {"actor_id": str (REQUIRED), "message": str (REQUIRED), "thread_id": str?, "metadata": dict?}
Response 200: {"thread_id": str, "status": str, "content": str?, "pending_approval": {...}?, "actions_taken": [...], "error": str?}

=== POST /api/v1/agent/invoke/stream ===
Content-Type: application/json → text/event-stream
Auth: NONE
Request: Same as /invoke
Response: SSE events: start, content/awaiting_approval, end, error

=== POST /api/v1/agent/approvals/{id}:approve ===
Content-Type: application/json
Auth: Optional Agent JWT (AGENT_JWT_ENFORCE=false)
Request: {"actor_id": str, "reason": str?}
Response 200: AgentInvokeResponse (same as /invoke)

=== POST /api/v1/agent/approvals/{id}:reject ===
Same as :approve

=== GET /api/v1/agent/approvals ===
Auth: NONE
Query: ?actor_id=str (optional filter)
Response 200: {"approvals": [...], "total": int}

=== GET /api/v1/agent/approvals/{id} ===
Auth: NONE
Response 200: {"approval_id": str, "tool": str, "args": dict, "permission_tier": str, "requested_by": str, "requested_at": datetime}

=== GET /api/v1/agent/threads/{id}/state ===
Auth: NONE
Response 200: {"thread_id": str, "status": str, "pending_approval": {...}?}

=== GET / ===
Response 200: {"name": str, "version": str, "status": str, "environment": str, "swagger_url": str, "redoc_url": str}

=== GET /health ===
Response 200: {"status": str, "version": str, "environment": str, "components": {...}, "timestamp": str}

=== GET /metrics ===
Content-Type: text/plain
Response: Prometheus metrics (process info, HTTP stats, DB connections, LLM inference duration)
