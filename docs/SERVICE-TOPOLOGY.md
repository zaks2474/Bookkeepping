### Backend API (FastAPI)
| Property | Value |
|----------|-------|
| **Canonical Port** | 8091 (port 8090 is DECOMMISSIONED) |
| **Health Endpoints** | `GET /health`, `/health/live`, `/health/ready`, `/health/startup`, `/health/detailed`, `/health/metrics` |
| **Health Response** | Structured JSON: `{status, database, outbox, version, uptime}` |
| **Startup Dependencies** | PostgreSQL (zakops DB, schema: zakops) — DSN gate validates canonical DB identity |
| **Discovery Variables** | `API_URL`, `BACKEND_URL`, `DEAL_API_URL`, `ZAKOPS_BACKEND_URL` |
| **Failure Detection** | Dashboard middleware returns JSON 502 on connection failure; Agent-api uses circuit breaker |
| **Runtime** | Docker container `zakops-backend-backend-1` |
| **Evidence** | `zakops-backend/src/api/shared/routers/health.py`, `zakops-backend/docker-compose.yml` |
